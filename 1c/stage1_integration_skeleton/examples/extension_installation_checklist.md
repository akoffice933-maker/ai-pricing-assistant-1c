# Чеклист установки расширения AI Pricing Assistant в 1С

## 1. Подготовка FastAPI

- [ ] FastAPI-сервис запущен.
- [ ] `GET /health` возвращает `status=ok`.
- [ ] `GET /model_info` возвращает версию модели.
- [ ] Если включён токен, он согласован с 1С.

## 2. Метаданные 1С

- [ ] Создано расширение `AI_Pricing_Assistant`.
- [ ] Создан регистр сведений `AI_РекомендацииПоЦенам`.
- [ ] Созданы константы `AI_Pricing_*`.
- [ ] Созданы общие модули `AIPricing*`.
- [ ] Создана обработка/форма `AI_РекомендацииПоЦенам`.
- [ ] Назначены права пользователям.

## 3. Адаптация источников данных

- [ ] Текущая цена корректно читается из регистра цен.
- [ ] Себестоимость корректно определяется.
- [ ] Остатки корректно определяются.
- [ ] Продажи за 30/90 дней корректно считаются.
- [ ] Данные конкурентов либо подключены, либо корректно отсутствуют.
- [ ] Акции и блокировки цен учитываются.

## 4. Функциональная проверка

- [ ] Рассчитана рекомендация по одному товару.
- [ ] В регистре сохранены JSONВход и JSONВыход.
- [ ] Рекомендация отображается в форме.
- [ ] Статус меняется на `Принята`.
- [ ] При отклонении сохраняется причина.
- [ ] Создаётся непроведённый документ `УстановкаЦенНоменклатуры`.
- [ ] Документ не проводится автоматически.

## 5. Безопасность

- [ ] API-токен не доступен обычным пользователям.
- [ ] JSON-аудит доступен только администраторам.
- [ ] AI не имеет права проведения документа.
- [ ] Массовое применение ограничено минимальной надёжностью.
