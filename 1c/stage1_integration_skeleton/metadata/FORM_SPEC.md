# Спецификация формы `AI_РекомендацииПоЦенам`

## Назначение

Форма показывает AI-рекомендации по изменению цен, позволяет рассчитать новые рекомендации, принять/отклонить их и создать непроведённый документ `УстановкаЦенНоменклатуры`.

## Реквизиты формы

| Имя | Тип | Комментарий |
|---|---|---|
| `Рекомендации` | ТаблицаЗначений | Таблица списка рекомендаций. |
| `ДатаНачала` | Дата | Фильтр. |
| `ДатаОкончания` | Дата | Фильтр. |
| `ФильтрСтатус` | Строка(30) | `Новая`, `Принята`, etc. |
| `ФильтрНоменклатура` | СправочникСсылка.Номенклатура | Фильтр. |
| `ФильтрВидЦены` | СправочникСсылка.ВидыЦен | Фильтр. |
| `МинимальнаяНадежность` | Число(4,2) | Фильтр confidence. |
| `БизнесЦель` | Строка(50) | Например `maximize_margin`. |
| `ТекущаяНоменклатура` | СправочникСсылка.Номенклатура | Для расчёта одного товара. |
| `ТекущийВидЦены` | СправочникСсылка.ВидыЦен | Для расчёта. |

## Колонки таблицы `Рекомендации`

- `ИдентификаторРекомендации`

## Команды формы

| Команда | Действие |
|---|---|
| `ОбновитьСписок` | Перечитать регистр `AI_РекомендацииПоЦенам`. |
| `РассчитатьДляТовара` | Собрать данные по текущему товару и вызвать `/skills/recommend_price`. |
| `РассчитатьПоГруппе` | Массовый расчёт по отобранным товарам. На MVP можно оставить как заглушку. |
| `ПринятьРекомендацию` | Статус `Принята`. |
| `ОтклонитьРекомендацию` | Запросить причину и установить статус `Отклонена`. |
| `СоздатьДокументЦен` | Создать непроведённый документ `УстановкаЦенНоменклатуры`. |
| `ОткрытьДокументЦен` | Открыть связанный документ, если он уже создан. |
| `ПоказатьJSON` | Показать JSON входа/выхода для администратора. |

## UX-правила

1. Рекомендации с `Надежность < 0.70` подсвечивать жёлтым/красным.
2. `УровеньРиска = high` подсвечивать красным.
3. Кнопка создания документа активна только для статусов `Новая`, `Просмотрена`, `Принята` и при наличии `РекомендованнаяЦена > 0`.
4. Документ создаётся непроведённым.
5. При отклонении причина обязательна.
6. JSON входа/выхода показывать только пользователям с ролью администратора AI.
