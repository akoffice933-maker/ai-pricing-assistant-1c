# Pilot plan

## Цель пилота

Проверить, что market-aware pricing даёт полезные рекомендации и улучшает принятие ценовых решений в 1С.

## Длительность

4 недели:

1. Неделя 1 — настройка и загрузка данных.
2. Неделя 2 — расчёт рекомендаций без применения.
3. Неделя 3 — применение части рекомендаций через согласование.
4. Неделя 4 — анализ фактического эффекта.

## Пилотная группа

Выбрать 20–100 позиций:

- 1–2 категории товаров;
- 1 категория услуг, если применимо;
- позиции с продажами за последние 90 дней;
- позиции с доступными рыночными индикаторами.

## KPI

| KPI | Цель |
|---|---:|
| Доля позиций с market_context | > 80% |
| Доля рекомендаций с confidence >= 0.7 | > 50% |
| Доля принятых рекомендаций | > 20% |
| Заполненность JSON-аудита | 100% |
| Создание документов только непроведёнными | 100% |

## Контрольная группа

Для оценки эффекта оставить часть похожих позиций без изменения цены.

## Метрики эффекта

- спрос/продажи до и после;
- выручка;
- валовая прибыль;
- маржа;
- отклонение факта от demand curve;
- причины отклонения рекомендаций.

## Результат пилота

Отчёт:

```text
- сколько рекомендаций рассчитано;
- сколько принято;
- какой ожидаемый эффект;
- какой фактический эффект;
- какие данные нужно улучшить;
- готовность к production.
```
