# Регистр сведений `AI_РыночныеИндикаторы`

Назначение: хранить индикаторы общего рынка, которые используются для построения кривой спроса относительно рынка.

## Тип

Регистр сведений, желательно периодический по месяцу/дню в зависимости от частоты обновления.

## Измерения

| Имя | Тип | Комментарий |
|---|---|---|
| `Период` | Дата | Период актуальности индикаторов. |
| `КатегорияРынка` | Строка(150) / справочник | Например `wireless_headphones`, `1c_implementation_services`. |
| `Регион` | Строка(50) | Например `LV`, `Riga`. |
| `КаналПродаж` | Строка(50) | `online`, `retail`, `marketplace`, `b2b_direct`. |
| `ТипПозиции` | Строка(30) | `product`, `service`, `subscription`, `project`. |
| `ИсточникДанных` | Строка(100) | Парсер, маркетплейс, CRM, ручная загрузка. |

## Ресурсы

| Имя | Тип | Комментарий |
|---|---|---|
| `MarketPriceMin` | Число(15,2) | Минимальная цена рынка. |
| `MarketPriceP25` | Число(15,2) | 25-й перцентиль. |
| `MarketPriceMedian` | Число(15,2) | Медиана рынка — главный якорь relative price. |
| `MarketPriceAvg` | Число(15,2) | Средняя цена. |
| `MarketPriceP75` | Число(15,2) | 75-й перцентиль. |
| `MarketPriceMax` | Число(15,2) | Максимальная цена. |
| `CompetitorCount` | Число(10,0) | Количество конкурентов. |
| `ActiveCompetitorCount` | Число(10,0) | Активные конкуренты. |
| `MarketDemandIndex` | Число(8,4) | 1.00 = обычный уровень спроса. |
| `SearchTrendIndex` | Число(8,4) | Индекс поискового интереса. |
| `LeadVolumeIndex` | Число(8,4) | Индекс лидов/заявок. |
| `CategoryViewsIndex` | Число(8,4) | Просмотры категории. |
| `PromoShare` | Число(6,4) | Доля рынка в промо 0..1. |
| `AvailabilityIndex` | Число(6,4) | Доступность рынка 0..1. |
| `SeasonalityIndex` | Число(8,4) | Сезонность. |
| `DataFreshnessDays` | Число(5,0) | Возраст данных. |
| `SourceCount` | Число(5,0) | Количество источников. |
| `CoverageScore` | Число(6,4) | Покрытие рынка 0..1. |
| `Confidence` | Число(6,4) | Надёжность индикаторов 0..1. |

## Реквизиты

| Имя | Тип | Комментарий |
|---|---|---|
| `Currency` | Строка(8) | Валюта рынка. |
| `RawDataJSON` | Строка неограниченной длины | Сырые данные источников. |
| `CalculationMethod` | Строка(250) | Метод расчёта индексов. |
| `LoadedAt` | Дата | Время загрузки. |
| `Comment` | Строка неограниченной длины | Комментарий. |

## Минимально обязательные поля

Для MVP нужны:

```text
MarketPriceMedian
MarketDemandIndex
PromoShare
AvailabilityIndex
SeasonalityIndex
DataFreshnessDays
Confidence
```
