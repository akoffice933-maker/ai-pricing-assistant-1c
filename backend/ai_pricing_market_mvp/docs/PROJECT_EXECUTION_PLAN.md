# План выполнения проекта

## Цель MVP

Перевести AI Pricing Assistant с прямого прогноза цены на правильную архитектуру:

```text
Market Context → Demand Curve → Price Optimization
```

## Выполнено в этой версии

- [x] FastAPI endpoint `/skills/forecast_demand_curve`.
- [x] FastAPI endpoint `/skills/optimize_price`.
- [x] FastAPI endpoint `/skills/recommend_price` как оркестратор.
- [x] Поддержка товаров и услуг.
- [x] JSON-схемы примеров товара и услуги.
- [x] Схема регистра `AI_РыночныеИндикаторы`.
- [x] BSL-модуль формирования `market_context`.
- [x] BSL-патч для формирования нового запроса из 1С.
- [x] Function calling schema для LLM.
